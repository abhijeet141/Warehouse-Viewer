# Net banners

Drop the real banner photos here (`.jpg`, `.jpeg`, `.png` or `.webp`).

They are hung automatically on the safety netting (F row, bays 1–24) in
filename order — e.g. `01-metro.jpg`, `02-warehouse.jpg`, … No code change
needed; just restart/reload the dev server after adding files.

While this folder has no images, stylized placeholder posters are shown.
